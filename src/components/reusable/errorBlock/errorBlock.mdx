# Error Block

Error block.

## Usage

### JS Import

```js
import '@kyndryl-design-system/shidoka-applications/components/reusable/errorBlock';
```

### HTML Tag

```html
<kyn-error-block></kyn-error-block>
```

## Mascot Images

Mascot image SVG assets can be found in and imported from [Shidoka Foundation](https://shidoka-foundation.netlify.app/?path=/story/foundation-image-assets--mascot).

### Slots

`kyn-error-block` has the following slots:

- **unnamed**: Slot for the error description.
- **image**: Slot for the error image.
- **actions**: Slot for the action buttons.
