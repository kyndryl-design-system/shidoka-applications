## Widget Sizes

Widget size definitons are provided. These can be imported and included in Layout deifinitions.

### Example

```js
import { WidgetSizes } from '@kyndryl-design-system/shidoka-applications/common/helpers/gridstack';

const Layout = {
  md: [
    {
      id: 'widget1',
      ...WidgetSizes.tower.md,
    },
  ],
  sm: [
    {
      id: 'widget1',
      ...WidgetSizes.tower.sm,
    },
  ],
};
```
