# Divider

Divider component.

## Usage

### JS Import

```js
import '@kyndryl-design-system/shidoka-applications/components/reusable/divider';
```

### HTML Tag

```html
<kyn-divider></kyn-divider>
```

## Primary Example

<Canvas>
  <Story of={DividerStories} />
</Canvas>

### Properties Reference

`kyn-divider` has following property:

- **vertical**: Renders the divider vertically. _Note_ : When enabled, the divider will span the full height of its parent container.

### HTML Tag's

```html
<!-- Horizontal divider -->
<kyn-divider></kyn-divider>

<!-- Vertical divider -->
<kyn-divider vertical></kyn-divider>
```
