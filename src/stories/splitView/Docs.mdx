## Usage

Split View is documented as a pattern composition, not a new shipped wrapper component.
Use existing components and your own pane content, widths, and backgrounds.

<Source
  language="html"
  code={`
<div class="split-view">
  <section class="pane pane--start">...</section>
  <div class="split-view__divider" role="separator" aria-orientation="vertical">
    <kyn-divider vertical drag-handle></kyn-divider>
  </div>
  <section class="pane pane--primary">...</section>
</div>
`}
/>
