## Usage

Use named slots for each pane and listen to `split-view-resize` when you need resize telemetry.

<Source
  language="html"
  code={`
<split-view-pattern
  panes="3"
  pane-1-label="Issue list"
  pane-2-label="Issue detail"
  pane-3-label="Code rail"
>
  <div slot="pane-1">...</div>
  <div slot="pane-2">...</div>
  <div slot="pane-3">...</div>
</split-view-pattern>
`}
/>
